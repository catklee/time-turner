⏳ Time-Turner

A Chrome extension that captures the hidden video links behind your Canvas / Leccap lectures, so you can save them to a permanent library or download them for offline studying — no more digging through Inspect Element.

⬇ Download the extension (.zip)
Small download · installs in about a minute

How to install

Time-Turner isn't in the Chrome Web Store, so you install it in Chrome's developer mode. It's a one-time setup:

1
Unzip the download Double-click time-turner-extension.zip in your Downloads folder. You'll get a folder — keep it somewhere permanent (e.g. your Documents folder), because Chrome loads the extension from that folder every time it starts.
2
Open Chrome's extensions page Type chrome://extensions into the address bar and press Enter.
3
Turn on Developer mode Flip the Developer mode toggle in the top-right corner of the page. Three new buttons appear.
4
Load the extension Click Load unpacked and select the unzipped Time-Turner folder. The extension appears in your list — you're done installing.
5
Pin it & use it Click the puzzle-piece icon next to the address bar and pin Time-Turner. Then open any Canvas/Leccap lecture, play the video for a few seconds, and click the Time-Turner icon — Save Link and Download Now buttons will be waiting for you.

Good to know

Why developer mode? Chrome requires it for any extension installed outside the Web Store. It's safe — it just means you're loading the code directly, and you can read every line of it in the folder you unzipped.

Do I need an account? Downloading lectures works with no sign-in at all. Signing in with Google (one click, no password) unlocks the synced library, so your saved links show up here on this website from any device.

Where do downloads go? Into a downloaded lectures folder inside your browser's downloads location.
